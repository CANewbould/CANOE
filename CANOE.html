Charles A Newbould's (Special) version of Open Euphoria (CANOE)

This document summarises the facilities so far available in this project, as stored in the various modules in the library folder.

address.e

  • address
    • Address(atom) --> address
    • copy(address,address,integer) --> address
    • free(address) --> void
    • poke_(address,object,flag) --> address

atom.e

  • atom (built-in)
    • abs(atom) --> atom
    • sign(atom) --> integer

boolean.e

  • boolean
    • FALSE
    • TRUE
    • iif(boolean,object,object) --> object

char.e

  • char
    • lower(char) --> char
    • s2(char) --> char
    • upper(char) --> char

clib.e

  • clib
    • Clib({string}) --> clib

crid.e

  • crid
    • Crid(string,clib,ints,integer) --> crid
    • fC(crid,ints) --> integer
    • vfC(crid,ints) --> void

creole.e

  • object (built-in)
    • write(object,integer) --> void
    • writef(object,string) --> void
    • prnt(object) --> void
  • string
    • display(string) --> flag

html.e

  • object (built-in)
    • write(object,integer) --> void
    • writef(object,string) --> void
    • prnt(object) --> void
  • string
    • display(string) --> flag

integer.e

  • integer (built-in)
    • mod(integer,integer) --> integer

ints

  • ints
    • Ints(integer,integer,integer) --> ints

io

  • filehandle
    • flength(filehandle) --> integer
    • flines(filehandle) --> {string}
    • fread(filehandle) --> string
    • fseek(filehandle,atom) --> boolean
    • fwhere(filehandle) --> integer
    • fwrite(filehandle,string) --> void
    • fwrites(filehandle,{string}) --> sequence
    • getCh(filehandle) --> integer
    • getString(filehandle) --> string
  • object (built-in)
    • write(object) --> void
    • writef(object,string) --> void
    • writefln(object,string) --> void
    • writeln(object) --> void

list.e

  • list
    • product(list) --> atom
    • sum(list) --> atom

sequence.e

  • sequence (built-in)
    • add2(sequence,object,flag) --> sequence
    • filter(sequence,string,object) --> sequence
    • filter_(sequence,string) --> sequence
    • FindAll(sequence,object) --> ints
    • fold(sequence,string,object) --> object
    • head_(sequence) --> object
    • init(sequence) --> sequence
    • last(sequence) --> object
    • map(sequence,string,object) --> sequence
    • map_(sequence,string) --> sequence
    • reverse(sequence) --> sequence
    • tail(sequence) --> sequence

string.e

  • string
    • s2c(string,integer) --> address
    • s2i(string) --> integer
    • split(string) --> sequence
    • startsWith(string,string) --> boolean
    • toLower(string) --> string
    • toUpper(string) --> string
    • trim(string,flag) --> string

totext.e

  • string
    • write(string,integer) --> void
    • display(string) --> flag
  • object (built-in)
    • writef(object,string) --> void
  • ints
    • writel(ints) --> void

warn.e

  • object (built-in)
    • warn(object) --> object
